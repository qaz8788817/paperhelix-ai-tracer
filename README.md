# 📊 PaperHelix AI Tracker - Academic Genealogy Canvas

> **A Chronological Citation Mapping and Academic Genealogy Tracker Utilizing Gemini 2.5 Flash to Deconstruct Structural Literature Evolutions.**

PaperHelix AI Tracker is an advanced desktop citation intelligence utility architected for hard-core researchers to instantly map the evolutionary timelines of specific scientific paradigms (such as dMRI noise processing models, ischemic stroke metrics, or adversarial image translations). By analyzing complex Introduction prose or standard References blocks, the background intelligence reconstructs chronological lineage chains, illustrating how algorithms pass down, branch out, or optimize across generations.

---

## ✨ Key Features

* **🧬 Chronological Lineage Extraction**: Automatically parses fragmented scientific citation nodes and structures them into an explicitly ordered JSON array filtered from past to present.
* **🎯 Progressive Dopamine Visual Timeline**: Renders cascading milestone cards cycling through signature macaron palettes (`#FFB3BA`, `#BAE1FF`, `#FDFDC9`) linked with handwritten generational down-arrows (`⬇️`).
* **✍️ Structural Word-Wrapping Interfaces**: Employs frictionless read-only `CTkTextbox` templates inside visual nodes, providing safe margin buffers for custom script typography while maintaining native clipboard copy accessibility.
* **🔍 Instant Conceptual Tagging**: Isolates core mathematical indicators and clinical domain terms (e.g., `#FA`, `#Residual_Learning`) below each historical epoch milestone card.

---

## 🛠️ Tech Stack & Dependencies

* **GUI Engine Shell**: `CustomTkinter` (Python 3.10+) - Responsive vertical scroll grids.
* **Semantic Parsing Engine**: `google-genai` (2025 Next-Gen Official Google SDK)
* **Model Selection**: `gemini-2.5-flash` (Optimized with strict nested Pydantic response schemas at a low 0.2 temperature for high empirical academic fidelity)
* **Storage Framework**: Real-time object serialization feeding into isolated runtime display components.

---

## 🚀 Getting Started

### Prerequisites

Configure the necessary visualization frameworks alongside the machine learning endpoint binaries inside your terminal terminal shell:

```bash
pip install customtkinter google-genai pydantic
```
